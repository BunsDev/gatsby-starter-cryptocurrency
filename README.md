## Install

- `git clone https://github.com/n8tb1t/gatsby-starter-cryptocurrency.git`
- `cd gatsby-starter-cryptocurrency`
- `yarn install`
- **Windows:** `sh .\bin\retrieve-documentation`
- **Linux:** `sh ./bin/retrieve-documentation`

## Start

- `gatsby develop`
